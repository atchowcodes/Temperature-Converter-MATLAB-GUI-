# Temperature-Converter-MATLAB-GUI-

A simple MATLAB GUIDE application for real-time temperature conversion between Celsius, Fahrenheit, and Kelvin.

<img width="677" height="734" alt="image" src="https://github.com/user-attachments/assets/154b0948-c1cd-46ca-9afe-07119893dfe1" />


## Features
- Convert between Celsius, Fahrenheit, and Kelvin
- Input validation with error message if no unit is selected
- Clear button to reset all fields
